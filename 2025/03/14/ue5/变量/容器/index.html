void Add(const T& Element); 将一个元素添加到集合中。如果元素已经存在，则不会重复添加。
void Remove(const T& Element);() 从集合中移除一个元素。如果元素不存在于集合中，则不会有任何影响。
bool Contains(const T& Element) const;() 检查集合中是否包含指定的元素
bool IsEmpty() const;() 检查集合是否为空。
int32 Size() const;() 获取集合中元素的数量
void Clear();() 清空集合中的所有元素
template<typename… Args> void Emplace(Args&&… Args);() 在集合中创建并添加一个元素。与Add方法类似，但如果元素已经存在，则不会重复添加
